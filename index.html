Videoaula - Implementação de uma API em Node.js

12 de junho, 2023

O Node-API-UTFPR é um projeto educacional que visa fornecer uma introdução prática e acessível ao desenvolvimento de APIs utilizando Node.js e Express. Com uma abordagem simplificada, nosso objetivo é garantir que os participantes tenham uma compreensão clara dos conceitos fundamentais e se sintam confiantes na implementação de APIs.

Neste projeto, forneceremos um CRUD (Create, Read, Update, Delete) básico como ponto de partida. Através de exemplos de código concisos e explicativos, abordaremos as principais etapas para criação de rotas, manipulação de dados, interação com o banco de dados e outras funcionalidades essenciais.

Caros alunos,

Gostaríamos de convidá-los para participar de uma videoaula especial sobre a implementação de uma API em Node.js. Nesta aula, exploraremos os conceitos fundamentais para construir e desenvolver uma API utilizando uma das linguagens mais populares para desenvolvimento web.

Apresentado por: João Matheus Bueno
